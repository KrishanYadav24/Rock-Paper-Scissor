# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using **HTML**, **CSS**, and **JavaScript**. The game allows a user to play against the computer with real-time score updates and result messages.

---

## Features

- Interactive UI with images
- User vs Computer gameplay
- Random computer choice generation
- Live score tracking
- Win/Lose/Draw result messages
- Responsive and clean design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## How It Works

1. User selects Rock, Paper, or Scissors.
2. Computer randomly generates a choice.
3. The winner is determined based on game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Scores update automatically.
5. Result message is displayed on screen.

---

## Game Logic

The JavaScript handles:

- Random computer selection
- Winner calculation
- Score updates
- DOM manipulation for live UI updates

---

## Setup Instructions

1. Download or clone the project.
2. Make sure the `images` folder contains:
   - `rock.png`
   - `paper.png`
   - `scissors.png`
3. Open `index.html` in your browser.
